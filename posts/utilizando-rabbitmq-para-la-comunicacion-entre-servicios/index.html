Utilizando RabbitMQ para la comunicación entre servicios

(Octubre 23, 2024), Categorias: Event-Driven Architecture RabbitMQ

Introducción

Cuando trabajamos con múltiples microservicios que necesitan comunicarse entre sí, RabbitMQ es una herramienta excelente para gestionar la mensajería entre ellos.

A continuación, enseñaré uno de los posibles flujos de trabajo básico utilizando esta cola de mensajería y como ejemplo usaré un sistema de comercio electrónico en el que tenemos un microservicio encargado de gestionar pedidos y otro encargado de notificar a los usuarios.
